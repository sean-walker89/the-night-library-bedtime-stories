## What is The Night Library?

The Night Library is a bedtime story app that produces tales in real time while staying entirely on-device on Mac Apple Silicon. It is built for quick, offline use when you want a storyteller that does not depend on accounts, API keys, or cloud services.

It combines local writing and illustration models so text and images can be created together. A live meter for reading and writing density helps set the rhythm as each page appears, and the next-page interaction remains responsive when children want to steer the story onward.

## Highlights

- Creates bedtime stories as they are being read
- Operates fully locally with no accounts or API keys
- Combines a local writer model with a local image model
- Includes procedural dream-canvas style illustration support
- Displays a live writing and reading density meter
- Keeps next-page responses quick when prompted by a child
- Continues offline after the app and local models are in place
- Designed around PrismML Bonsai and a local AI workflow

## How to Use It

1. Open The Night Library on a Mac Apple Silicon device.
2. Start or select a bedtime story session.
3. Let the local writer produce the next section of the tale in real time.
4. Use the next-page input to guide where the story goes next.
5. Observe the density meter and illustration layer as the story expands.
6. Keep using it offline for the rest of the session, assuming the required local components are already available.

Example workflow:

- open the app
- start a new story
- wait for the first page to appear
- tap or type a prompt to continue
- read the illustrated result aloud
